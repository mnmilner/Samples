Before trying to run this sample you must do the following:

1) Provision a mobile service in Azure
	Topic "Create a new mobile serivce" at:
	https://azure.microsoft.com/en-us/documentation/articles/partner-xamarin-mobile-services-android-get-started/

2) Create a table in your mobile service named "LegoSet"
	https://msdn.microsoft.com/en-us/library/azure/jj193162.aspx
	
3) Update the URL and application key in the LegoManagerService.cs file in the LegoManager project.
